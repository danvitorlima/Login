# Sistema de Login em Contexto de Jogo (Offline)

## Descrição do Projeto

Este projeto é um **sistema de login** criado para funcionar em **contexto de jogo**.
Ele funciona **offline**, sem internet e sem banco de dados.

O login é apenas uma **simulação**, feita localmente no navegador.
O objetivo é estudar e aplicar conceitos de **Interação Humano-Computador (IHC)**.

O visual do sistema é moderno, com cores escuras e estilo gamer, para passar sensação de **segurança**, **imersão** e **jogo**.

---

## Objetivo do Projeto

O objetivo é criar um sistema simples e funcional, focado em:

* Fácil de usar
* Fácil de entender
* Feedback rápido ao usuário
* Experiência agradável
* Simulação segura de login

---

## Funcionamento do Sistema

O usuário digita **email** e **senha**.
O sistema valida os dados localmente.
Não existe banco de dados.
Não existe conexão com internet.

O sistema mostra mensagens de erro ou sucesso usando **animações e efeitos visuais**.

---

## Requisitos Funcionais Atendidos

* Interface de login
* Validação local dos dados
* Mensagens de erro e sucesso
* Feedback visual ao usuário
* Funcionamento offline
* Simulação de autenticação

## Metas de Usabilidade

### Fácil de lembrar como usar

Os campos são simples e organizados.
O usuário entende rápido como usar.

### Fácil de entender

Textos claros e feedback visual ajudam o usuário durante o uso.

### Útil

O sistema cumpre sua função de login dentro de um jogo.

### Seguro (percepção do usuário)

Cores escuras e animações suaves passam sensação de segurança.

### Eficiente

O sistema responde rápido, sem passos desnecessários.

## Metas de Experiência do Usuário

As metas escolhidas foram:

* **Divertido**
  Visual inspirado em jogos deixa a experiência mais interessante.

* **Agradável**
  Layout limpo e animações suaves não cansam o usuário.

* **Motivador**
  O feedback positivo incentiva o usuário a continuar.

* **Esteticamente apreciável**
  Design moderno e estilo gamer aumentam o interesse.

* **Satisfatório**
  O usuário entende claramente se o login funcionou ou não.
